# Design summary

- Design concept: Company comparison journal
- Layout personality: newspaper-inspired
- Typography: Newsreader headings with Source Serif 4 body text
- Color palette: primary #3f3f46, secondary #d4d4d8, accent #ca8a04, background #fafafa, text #18181b, muted #71717a
- Shape language: Sharp corners
- Visual motif: newspaper columns and comparison rules
- Unique layout choices: first sections use Buyer journal, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
